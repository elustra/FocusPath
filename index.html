4 Core Features

From visual audit
to logic authoring.

Every tool runs on local math — no AI API, no data sent externally. 100% private and instant.

Visual Focus Mapper
Tab Order · Screen Reader Path

Click elements in the exact order a keyboard or blind user should navigate. The plugin draws a numbered blue "thread" connecting every step. Drag to re-order. Export the sequence as a developer handoff annotation.

💡 Competitors like Microsoft's A11y Focus Order haven't been updated in years and crash on complex files. FocusPath is stable, fast, and handles any layer depth.
Audio Narrator
USP · Talking Prototype · Demo Tool

Press Play. Your computer's native voice reads the design out loud in the exact order you mapped — using ARIA labels where saved, text content where not. Turn a flat frame into a talking prototype in one click.

💡 No competitor does this. Most tools only show data. FocusPath lets stakeholders and clients hear the brand experience. It is the most powerful demo tool in any accessibility pitch.
Contrast Healer
WCAG AA · AAA · Mathematical Fix

Select a frame. FocusPath scans every text layer, calculates the WCAG contrast ratio against its background, and flags failures. Click Heal — the plugin mathematically finds the nearest compliant colour and applies it directly.

💡 Every other tool gives you a red X. FocusPath gives you the answer. No AI, pure luminance math — fixing contrast in one click without breaking your brand palette.
ARIA Sticky Notes
Dev Handoff · Zero Redlines · pluginData

Select any layer. Type its aria-label, choose its ARIA role, add a description, and mark it hidden if needed. All data is stored in Figma's pluginData — visible in Dev Mode. Export the entire page as a JSON spec.

💡 Eliminates messy red-line annotation frames. Developers read ARIA data directly from the Figma file, cutting handoff time by hours.
Process

Author. Not Audit.

Accessibility built into the design process — not bolted on at the end.

Step 01
Map the Tab Order

Select layers in the order a keyboard user should navigate them. The Focus Mapper draws a numbered thread directly on the canvas.

Step 02
Hear the Design

Click Play. The Audio Narrator reads every element in order — using ARIA labels where saved — via your system's native voice engine.

Step 03
Heal Contrast

Scan any frame for WCAG failures. One-click Heal finds the mathematically nearest compliant colour and applies it without breaking your palette.

Step 04
Attach ARIA Labels

Annotate every icon, button, and interactive element with ARIA data. Export a developer-ready JSON spec. No redline frames needed.

Contrast Healer

Stop telling. Start fixing.

Other tools give you a red X. FocusPath gives you the corrected colour — applied to your Figma layer in one click.

The Contrast Healer scans every text layer in your selection, walks up the parent tree to find the background colour, computes the WCAG relative luminance ratio, and mathematically darkens or lightens the text colour by the smallest possible amount until the ratio passes. No approximations, no AI guesswork.

  • →WCAG AA (4.5:1) for normal text
  • →WCAG AA Large (3:1) for 18px+ text
  • →WCAG AAA (7:1) for strict compliance
  • →Heal All Failing — one button, entire frame
